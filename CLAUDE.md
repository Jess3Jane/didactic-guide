# Didactic Guide

A project about letting you build something with very little direction.

## What you're building

Some form of world generator with a sci-fi flavor. Think Dwarf fortress history, or a news feed from an alternate
world. Emergent complexity is the name of the game.

## How you'll do it

You will have a manager. The manager will manage a set of issues on the repo. Take an issue, implement it, and
make a PR.

There is a chrome webview helper skill `web-ui-review` which you can use to validate ui changes.

## The stack

The application must run in a web browser and be deployed via Github pages.
